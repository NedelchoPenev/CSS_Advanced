About the course

The course "CSS Advanced" takes a deeper insight into the modern front-end design and development. Participants will have the opportunity to build on their knowledge of web development pages with animation, icons, and grid for adaptive rendering of the page on devices with different screen sizes (desktop, tablet, phone). Important an ability to work efficiently is building and following a clear workflow to include Contemporary tools to improve productivity, which students will be able to get acquainted with through their application in practical project.  Assessment of the course is based on the homeworks and practical exam.

Skills you will acquire

  • Building an efficient workflow
  • Work with preprocessors and postprocessors
  • Animate elements and transitions through CSS
  • Tracing the layout of customizable pages
  • Follow style guides and template libraries
  • Using tools to improve workflow
